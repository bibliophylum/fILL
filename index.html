Welcome to fILL

fILL is the interlibrary loan management system in use by the public libraries in the province of Manitoba, Canada.

An implementation of fILL will be a single centrally-hosted web site (Linux, Apache, Perl, and PostgreSQL); clients - that is, interlibrary loans staff at participating libraries - simply need a Javascript-enabled browser.

Parallel searching of z39.50 targets

fILL uses the excellent GPL-licensed Pazpar2 z39.50 middleware from Index Data (Pazpar2 from IndexData) to provide parallel searching of multiple library targets quickly, with merging, relevance ranking, record sorting, and faceted results.

Interlibrary loan life-cycle management

fILL handles the entire life cycle of interlibrary loans, from discovery through request, response, and return of the item. Life cycle management is based on the message types and states in ISO 10161 (Interlibrary Loan Application Protocol Specification). ISO 10161

Simple, powerful client-side

The client side uses jQuery and the fantastic DataTables plugin (jQuery DataTables) to provide sortable, searchable lists throughout the application. Almost everything in the system is presented to the user in these powerful tables - current requests, pull-lists, requests waiting for responses, etc.
